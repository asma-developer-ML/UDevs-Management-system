# U Devs Management System (FRONTEND)

## Description
U Devs Management System is a responsive frontend web interface built using HTML, CSS, and Bootstrap 5.  
The project focuses on clean layout design, responsiveness, and proper use of Bootstrap components.

## Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript (basic)

## Features
- Responsive navbar  
- Hero section  
- Bootstrap cards and modals  
- Mobile, tablet, and desktop responsive design  
- Footer section

## How to Run
Open `index.html` in any web browser.
